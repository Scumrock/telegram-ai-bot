## Сборка приложения

Перед тем, как собирать приложение, необходимо выбрать подходящий 
образ, заменив 
```
ultralytics/ultralytics:latest-arm64
```
на `amd-64` или `gpu` архитектуру.

После этого следует выполнить команду 
```
docker compose build
```

## Запуск приложения

```
docker compose up -d
```
